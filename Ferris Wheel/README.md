# Ferris Wheel
This project is a simple representation of a Ferris wheel using HTML and CSS. It creates a spinning wheel with multiple cabins.

## Getting Started
To view the Ferris wheel, simply open the `index.html` file in a web browser.

## Built With
- HTML
- CSS
